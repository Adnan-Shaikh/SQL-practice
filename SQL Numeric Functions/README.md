---

# Functions Covered

## ROUND()

Used to control decimal precision.

Example:

Financial reports where revenue values should display clean decimals.

---

## CEIL()

Rounds values upward.

Used in scenarios where partial values require a complete unit.

Example:

Calculating required pallets.

---

## FLOOR()

Rounds values downward.

Used when only complete units should be counted.

Example:

Finding complete boxes or production cycles.

---

## MOD (%)

Returns the remainder after division.

Used for:

- Splitting revenue
- Finding leftover values
